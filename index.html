git init: para iniciar el repo vim index.html git add ./*: creo archivos luego los agrego al control de git git commit -m "primer commit": cierro "la cajita" con ese archivo y los cambios efectuados. git remote add origin https://github.com/rodmontiel/acamica.git: sincronizo mi repo local con el remoto. git status: verifico estado de git git push -u origin master (o solamente git push): pusheo mi repo local al remoto git pull -u origin master (o simplemente git pull): me traigo los cambios del repo remoto a mi repo local asi quedamos sincronizados. git clone https://blabla/acamica.git: clono un repo remoto localmente.
